## 语法
### 数学模块
#### 空格

| 效果       | 语法        | 具体宽度               |
| -------- | --------- | ------------------ |
| 紧贴       | a\\! b    | 缩进$\frac{1}{6}$m宽度 |
| 正常       | ab        | 正常                 |
| 小空格      | a\\, b    | $\frac{1}{6}$m宽度   |
| 中等空格     | a\\; b    | $\frac{2}{7}$m宽度   |
| 大空格      | a\ b      | $\frac{1}{3}$m宽度   |
| quad空格   | a\quad b  | 一个m的宽度             |
| 两个quad空格 | a\qquad b | 两个m的宽度             |
